# ExecDock

xterm + node-pty + websocket

## Installation   

```
pnpm install
```

## Getting Started

```
pnpm run dev
```

## Generate production resources

```
pnpm run build
```

and

```
pnpm run serve
```

Open your browser http://localhost:8999
